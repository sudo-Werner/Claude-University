# Design assets from Claude design

Drop the Claude design exports here so the engine can be built against them.

- `tokens.md` — design-tokens spec (hex colours, type scale, spacing, radii, component states)
- `dashboard.png`, `dashboard-mobile.png` — Dashboard screenshots (desktop ~1280px, phone ~390px)
- `lesson.png`, `lesson-mobile.png` — Lesson flow screenshots (desktop, phone)
- `reference/` — raw HTML/CSS reference for each screen (`dashboard.html`, `lesson.html`, shared CSS)

Screenshots are the visual ground-truth; the HTML/CSS is what the real components are rebuilt from.
